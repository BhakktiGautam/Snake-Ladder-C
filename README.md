#  Snake and Ladder Game in C

## 📌 Overview
This project is a console-based implementation of the classic Snake and Ladder game developed using the C programming language. It supports two-player gameplay with an interactive menu and real-time board updates.

---

## 🎮 Features
- 🎲 Dice roll simulation using random numbers
- 👥 Two-player turn-based gameplay
- 🐍 Snake logic (move down)
- 🪜 Ladder logic (move up)
- 📊 Live scorecard display
- 🧩 Dynamic board printing (zig-zag pattern)
- 🚫 Boundary check (no move if position > 100)
- 📋 Menu-driven interface

---

## 🧠 Concepts Used
- Arrays (for snakes and ladders mapping)
- Functions and modular programming
- Loops and conditional statements
- Random number generation (`rand()` and `srand()`)

---

## 🚀 How to Run

### Step 1: Compile

gcc snake_ladder.c -o game
./game
📸 Sample Menu
1. Start Game
2. Print Board
3. Print Scorecard
4. Exit
